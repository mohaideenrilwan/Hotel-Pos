📋 Bluetooth Printer Setup
1. Pair printer with your device via OS Bluetooth settings first
2. Tap Scan & Connect — select your printer from the list
3. Works with any ESC/POS thermal printer (58mm & 80mm)
4. Compatible printers: GOOJPRT, Xprinter, Rongta, Epson TM, etc.
5. Requires Chrome / Edge on Android or Desktop
⚠ iOS Safari does NOT support Web Bluetooth

📋 Google Sheets Setup
1. Go to console.cloud.google.com
2. Create project → Enable Google Sheets API
3. Create OAuth 2.0 Client ID (Web Application)
4. Add your site URL to Authorised Origins
5. Copy Client ID → paste above
6. Create a Google Sheet → copy its ID from URL
7. Click Sign In with Google → then Create Sheet Headers
8. Sheet columns auto-created: Products | Bills | Returns
